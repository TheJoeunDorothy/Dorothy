# Dorothy

Flutter를 사용하여 IOS용 모바일 어플리케이션 제작.  
사용자가 찍은 얼굴 사진 데이터를 이용하여 딥러닝 모델을 통해 퍼스널컬라와 얼굴 나이를 예측해준다.
